Kubedge

Personal, Portable Edge Network and Lab.

Kubedge is personal and portable edge cloud. The key concept is to leverage the three network interfaces available on each PI:

  • 1 GB Ethernet
  • Wifi
  • Bluetooth.

The PI are interconnected together using the local switch. The top/master PI is acting as a NAT/Router for the rest of the PI. This main advantage that when you move your lab from home to work, all the IPs of the cluster stay identical.

This page and web site is still work in progress.
